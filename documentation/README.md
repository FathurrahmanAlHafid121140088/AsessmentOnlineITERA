## Kontribusi

Saat menambahkan dokumentasi baru:
1. Tentukan kategori yang sesuai
2. Letakkan file di folder kategori yang tepat
3. Update README.md di folder kategori tersebut
4. Gunakan format Markdown yang konsisten
5. Sertakan tanggal dan author jika diperlukan

## Konvensi Penamaan File

- Gunakan UPPERCASE untuk file dokumentasi utama
- Gunakan underscore (_) sebagai separator
- Format: `TOPIK_DESKRIPSI_TIPE.md`
- Contoh: `MENTAL_HEALTH_BACKEND_CODE.md`
